# Feature Modules

Este directorio se organiza por feature. Cada módulo contiene sus capas internas y su i18n local.

Estructura base:

- `api/`
- `application/`
- `domain/`
- `infrastructure/`
- `ui/`
- `i18n/` (`es-MX.json`, `en-US.json`)

Módulos iniciales:

- `leads`
- `clients`
- `suppliers`
- `itinerary`
- `commissions`
- `financials`
- `messaging`
- `dashboard`
- `management`
